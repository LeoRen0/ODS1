Acerca de este proyecto

🌍 🍽️ 🌱

Este sitio web es un proyecto educativo interactivo creado con HTML, CSS y JavaScript.

El objetivo es aprender desarrollo web mientras se trabaja el tema del ODS 2: Hambre Cero, enfocado en eliminar el hambre mundial.

El proyecto integra elementos visuales como carruseles, animaciones, tarjetas informativas y efectos interactivos para mejorar la experiencia del usuario.

También busca concienciar sobre la importancia de la alimentación sostenible y la reducción del desperdicio de alimentos.
